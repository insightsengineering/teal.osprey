# `teal.osprey`: `teal` modules for 'osprey'

`teal.osprey` provides community contributed
[`teal`](https://cran.r-project.org/package=teal) modules of the
analysis functions from `osprey` package. See package
[website](https://insightsengineering.github.io/osprey/). This enables
`teal` app developers to easily create applications making use of the
`osprey` analysis functions.
