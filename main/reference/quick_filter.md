# Utility function for quick filter

Utility function for quick filter

## Usage

``` r
quick_filter(filter_opt, ANL)
```

## Arguments

- filter_opt:

  vector of string names of flag variable to filter (keep Y rows only)

- ANL:

  input dataset

## Value

a filtered dataframe
